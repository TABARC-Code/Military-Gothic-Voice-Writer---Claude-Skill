## Extended description (for repository README header / pinned description)

**military-gothic-voice** is a structured Claude skill system that applies a specific and demanding prose register to any fiction you throw at it. The voice is military gothic — formally controlled, psychologically interior, physically precise, and carrying the permanent weight of things that already happened before the scene started.

It isn't a prompt template. It's a complete behaviour specification: seven sub-skills with sequential thinking procedures, bidirectional data contracts between every component, a four-layer violation taxonomy for revision work, and style rules derived from forensic corpus analysis rather than intuition.

The skill covers generation, revision, and diagnosis. In generation mode, it configures a sub-skill stack from the scene type you specify, drafts the prose, audits thematic embedding, and runs a full style gate before returning anything. In revision mode, it runs a structural > voice > mechanics > vocabulary scan, flags violations with location and severity, preserves what's working, and applies corrections in the correct order. In diagnosis mode, it does the scan without touching the prose — useful when you want to understand what's broken before you decide what to do about it.


Built and maintained by **TABARC-Code**.
